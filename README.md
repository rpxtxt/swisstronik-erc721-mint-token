# swisstronik-erc721-mint-token

## Installation

1. Clone this repository:

    ```sh
    git clone https://github.com/username/swisstronik-erc721-mint-token.git
    cd swisstronik-erc721-mint-token
    ```

2. Install dependencies:

    ```sh
    npm install
    ```

    or

    ```sh
    yarn install
    ```
